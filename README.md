
# PROYECTO FINAL

Este repositorio contiene el código fuente de una página web estática o un proyecto de interfaz de usuario. Fue desarrollado con el objetivo principal de aplicar y consolidar conocimientos fundamentales en las tecnologías esenciales del desarrollo web y el control de versiones, la puedes visualizar en el siguiente link: https://saraiceleste.github.io/Pagina-web-Dev-f.github.io/

## Tecnologías empleadas:HTML,Css, github, git.

El proyecto fue construido utilizando un conjunto de herramientas estándar de la industria, asegurando una estructura robusta y un control de desarrollo eficiente:

 - HTML5:	Utilizado para definir la estructura semántica del contenido de la página web.
 - CSS3:	Aplicado para la presentación visual (diseño, colores, tipografía y layout).
 - Git:	Sistema de control de versiones distribuido para rastrear cambios en el código de manera eficiente.
 - GitHub:	Plataforma de hosting que aloja el repositorio de Git, facilitando la colaboración y el despliegue del proyecto.## Paleta de colores
